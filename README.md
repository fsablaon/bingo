# Bingo project

Implement a game of Bingo

Bingo has
- Dealer
- Players
- Tracks the winning players

Dealer
- Deals a Cue

Player has a
- Name
- Ticket
- Checkers (a behaviour to check if they have a winning ticket)

Ticket has
- 2 dimentional sparse matrix of nodes (numbers)
- can mark when number is called
